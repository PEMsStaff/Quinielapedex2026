Para subir a GitHub Pages:
1. Descomprime este ZIP.
2. En tu repositorio, usa Add file > Upload files.
3. Arrastra index.html, Logo.png y la carpeta img completa.
4. Commit changes.
5. En Settings > Pages usa Deploy from branch, main, /root.
